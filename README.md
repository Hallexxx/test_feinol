## Pré-requis

Avant de lancer le projet, il faut avoir installé :

- Node.js
- npm

Vérification :

node -v
npm -v

## Installation

Cloner le dépôt :

git clone URL_DU_DEPOT

Puis installer les dépendances :

npm install

## Lancer le projet

Démarrer l’application en mode développement :

npm start

Puis ouvrir :

http://localhost:3000

## Lancer les tests

Exécuter les tests avec :

npm test

## Générer la documentation JSDoc

npm run jsdoc

La documentation sera générée dans le dossier :

/docs

## Build production

Créer une version production :

npm run build
